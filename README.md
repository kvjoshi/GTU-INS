# GTU-INS
GTU Information and Network Security ( INS ) in python .


Sample Inputs:
1. Caesar cipher:
          Mode : "encrypt" / "decrypt" |
          Message :  hello |
          Key: 5 
    
2. MonoAlphabetic Cipher: (Key is not dynamic)  |
         Mode : "encrypt" / "decrypt" |
         Message : Hello
    
3. Play-Fair Cipher:
         Mode : "encrypt" / "decrypt" |
         Message: hello |
         Key: world
    
4. Polyaphabetic Cipher:
         Mode : "encrypt" / "decrypt" |
         Message :Hello World
    
5. Hill Cipher:
         Mode : "encrypt" / "decrypt" |
         Key: HIFF |
         Message: hello
    
6. Diffe-Hellman:
      Any numeric value.
